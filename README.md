store front website for laptop and phones and accessories 
responsive , dynamique and firebase as backend and simple dashboard to add or remove or update products and prices and coupons and calculating net worth daily after sale and registering all products with buying price to know how much net profit and finally the ability to share the products easily on social media

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/smart-tech-store.git
cd smart-tech-store
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
